# Cisco Packet Tracer Projects

## Enterprise Campus Network

This project demonstrates the implementation of a small enterprise campus network using Cisco Packet Tracer. It was built as part of my CCNA practice to strengthen my understanding of enterprise networking concepts.

## Technologies Implemented

- VLAN Configuration
- Inter-VLAN Routing (Router-on-a-Stick)
- OSPF Dynamic Routing
- Spanning Tree Protocol (STP)
- VLSM Subnetting
- IPv4 Addressing
- 802.1Q Trunking
- Cisco IOS Configuration
- Network Troubleshooting

## Network Topology

Departments included:

- VLAN 10 – HR
- VLAN 20 – IT
- VLAN 30 – Sales

## Skills Demonstrated

- Enterprise Network Design
- Layer 2 Switching
- Layer 3 Routing
- Dynamic Routing
- VLAN Segmentation
- IP Address Planning
- Cisco Packet Tracer
